# UI Pattern Bootstrap Template

This repo is copied as plain files into a new UI project. The copy owns every file.

First steps:

```bash
pnpm install
pnpm run init -- <project-name>
pnpm run verify
```

The architecture contract is `den:patch/rusty-view-ui-architecture-pattern`; this template is the executable plumbing for that pattern.

Keep `template-manifest.json` with the copy. Record deliberate deviations in that file after adding an ADR under `docs/adr/`.
